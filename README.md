# NBFC Loan Origination System

Run locally:

```powershell
cd C:\Users\Fin-secadmin\Documents\NBFC_LOS
python app.py
```

Open [http://127.0.0.1:8000](http://127.0.0.1:8000)

Project includes:

- Python backend in `app.py`
- SQLite persistence in `nbfc_los.db` created automatically
- Frontend template in `templates/index.html`
- Static UI assets in `static/`
